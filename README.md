# 💰 Expense Tracker

A simple command-line based Expense Tracker built with Python that helps you record and manage your daily expenses.

---

## 📌 About the Project

This project was built to practice Python fundamentals like file handling, functions, and loops. It stores all expense data in a CSV file so your data is saved even after closing the program — no database required!

---

## ✨ Features

- ➕ Add a new expense with category, description, and amount
- 📋 View all recorded expenses in a clean table format
- 💵 View total amount spent
- 📊 View spending broken down by category
- 💾 Data saved automatically to a CSV file

---

## 🛠️ Tech Stack

- **Language:** Python 3
- **Modules used:** `csv`, `os`, `datetime` (all built-in — no installation needed!)

---

## ▶️ How to Run

1. Make sure Python is installed on your system
   ```bash
   python --version
   ```

2. Clone this repository
   ```bash
   git clone https://github.com/yourusername/expense-tracker.git
   cd expense-tracker
   ```

3. Run the program
   ```bash
   python expense_tracker.py
   ```

4. Use the menu to add and view your expenses!
